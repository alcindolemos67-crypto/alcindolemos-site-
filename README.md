# Alcindolemos — Loja de Músicas (Next 14 + Mercado Pago)
- Catálogo com prévia
- Checkout Mercado Pago
- Download protegido (token por 10 min)

## Como rodar
1. `npm install`
2. `.env.local` com `MP_ACCESS_TOKEN`, `SITE_URL`, `JWT_SECRET`
3. `npm run dev`

## Deploy Vercel
- Importar repo → adicionar variáveis (All Environments) → Deploy → Redeploy após ajustes.
